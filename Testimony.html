Testimony

Saya bersyukur bisa mengikuti project The Pieces ini, Project Spiritual Formation ini membantu saya untuk melihat diri saya lebih dalam mengenai apa saja yang membentuk manusia lama saya, seperti sifat atau karakter saya yang banyak dibentuk dari pengalaman saya dan pembentukan keluarga saya. Project ini juga mambawa saya untuk melihat bagaimana solusi untuk mengubah manusia lama tersebut ke dalam manusia baru di dalam Kristus. Saya bersyukur setelah mengikuti project ini saya bisa semakin memahami diri saya lagi dan terus berlatih untuk serupa dengan Kristus.

Theofilus Soleman Youth Pastor

Yang paling berkesan dari project spiritual formation ini adalah dimana kita belajar untuk menggali kembali hal hal dimasa lalu yang selama ini mungkin terpendam di bawah alam sadar. Namun dengan belajar untuk membuka diri, mengingat lagi hal hal dimasa lalu yang menganggu kehidupan saya secara spiritual. Saya melihat ada banyak sekali perubahan dan saya merasakan juga ada kelepasan dan kelegaan dan ini menolong saya juga makin bertumbuh semakin dekat kepada Tuhan.

Davina Junus Part-Time Pastor
